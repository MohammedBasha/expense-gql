# Expense Tracker with GraphQL

A full-stack Expense Tracker application that enables users to securely manage their financial transactions through a modern GraphQL API.

## Features

- Secure user authentication with session-based authentication
- Create, update, and delete expense transactions
- GraphQL API built with Apollo Server
- GraphQL queries and mutations for efficient data management
- Relationship-based GraphQL schema
- Modular GraphQL architecture using merged type definitions and resolvers
- Apollo Client for optimized data fetching and caching
- Responsive user interface built with React and Tailwind CSS
- MongoDB database integration with Mongoose
- Scheduled background tasks using Cron Jobs

## Tech Stack

### Backend

- Node.js
- Express.js
- Apollo Server
- GraphQL
- MongoDB
- Mongoose
- Passport.js
- Express Session
- connect-mongodb-session
- bcryptjs

### Frontend

- React
- Apollo Client
- Tailwind CSS

## Overview

The application provides a secure and scalable solution for tracking expenses and managing financial transactions. It combines a GraphQL backend with a modern React frontend, using Apollo Client for efficient communication and caching. The backend follows a modular architecture that keeps GraphQL schemas and resolvers organized, making the project easier to maintain and extend.
